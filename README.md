## How to Run

```bash
$ roscore
$ cd ~/catkin_ws
$ source devel/setup.bash
$ roslaunch json_prolog json_prolog.launch
$ cd ~/catkin_ws/src/knowrob/json_prolog/src/json_prolog 
$ python2 main.py
```

or just

$ cd scripts
$ bash run.sh

## Requirements

- tensorflow==1.8.0
- gym
- ros
- knowrob
- python 2.7
- imageio
- pandas
